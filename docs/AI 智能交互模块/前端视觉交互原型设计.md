### 1.4 动效曲线

| Token | 值 | 用途 |
|---|---|---|
| `--ease-standard` | `cubic-bezier(.4,0,.2,1)` | 抽屉滑入 / 按钮 |
| `--ease-emphasized` | `cubic-bezier(.2,0,0,1)` | FAB 弹出 |
| `--duration-fast` | 150ms | 颜色 / 透明度 |
| `--duration-normal` | 240ms | 抽屉 / 折叠 |
| `--duration-slow` | 400ms | 全屏切换 |

动画规范：

- **FAB 呼吸**：`scale(1) → scale(1.05) → scale(1)`，循环 2.4s，透明光晕同步扩散。
- **三点 thinking**：3 个圆点依次 `opacity 0.3 → 1 → 0.3`，相位差 0.2s，循环 1.4s。
- **打字机光标**：`▍` 高 1em，`opacity 0/1` 闪烁，循环 1s。
- **工具卡旋转图标**：`rotate(0 → 360deg)` 线性 1s。

## 3. FAB 浮动入口原型

### 3.1 静态视觉（4 态）

```
┌─────── 默认态 ───────┐    ┌─────── Hover ────────┐
│                      │    │                       │
│        ╭────╮        │    │       ╭─────╮         │
│        │ ✨ │        │    │       │  ✨ │         │
│        ╰────╯        │    │       ╰─────╯         │
│   56×56  渐变实底    │    │   60×60  阴影加深     │
│                      │    │   tooltip:「智能助手  │
│                      │    │   (Ctrl+/)」          │
└──────────────────────┘    └───────────────────────┘

┌────── Active(点击) ──┐    ┌──── 收起态(抽屉开) ──┐
│        ╭────╮        │    │        ╭────╮        │
│        │ ✨ │ scale  │    │        │ ✕  │        │
│        ╰────╯ 0.95   │    │        ╰────╯        │
│                      │    │   保持 56×56          │
│   100ms 反馈         │    │   渐变切换为 #64748b  │
└──────────────────────┘    └──────────────────────┘
```

- **位置**：`position: fixed; right: 24px; bottom: 24px; z-index: 1000;`
- **形状**：`border-radius: 50%; width: 56px; height: 56px;`
- **背景**：`linear-gradient(135deg, var(--ai-gradient-from), var(--ai-gradient-to));`
- **图标**：默认使用 Element Plus Icon `MagicStick`（或 ChatDotRound 备选），白色 24px。

### 3.2 微动画

- **呼吸光晕（默认态）**：FAB 后方一层 `box-shadow: 0 0 0 var(--ai-aura)` 在 `0px → 14px → 0px` 间循环 2.4s。
- **点击反馈**：`scale: 1 → 0.95 → 1`，120ms。

### 3.3 快捷键反馈

按下 `Ctrl + /` 时：
- FAB 闪一下蓝色高亮（80ms 透明度变化）
- 抽屉滑入

按下 `Esc` 时（抽屉打开状态）：抽屉滑出。

## 5. 会话列表组件原型

### 5.1 列表项设计

```
┌──────────────────────────────────────────────┐
│ ▍生产计划查询...               🗑           │ ← 选中态：左侧 3px 蓝条
│   3 分钟前 · ● ACTIVE                        │   背景 #e8efff
├──────────────────────────────────────────────┤
│  设备焊接机器人查询              hover→ 🗑  │
│  1 小时前 · ● ACTIVE                         │
├──────────────────────────────────────────────┤
│  焊接 BOM 创建                              │
│  昨天 16:20 · ○ ARCHIVED                     │ ← 灰点
└──────────────────────────────────────────────┘
```

- 标题：超过 20 字符截断为 `...`，hover 显示完整 tooltip
- 时间：相对时间（< 1min "刚刚"，< 1h "X 分钟前"，< 24h "X 小时前"，< 7d "昨天 HH:mm"，否则 `YYYY-MM-DD`）
- 删除图标：`Delete` 仅 hover 时显示，点击触发 ElMessageBox

### 5.2 交互流

#### 新建
```
点击「+ 新建会话」
  → POST /ai/chat/sessions {}
  → 列表头插入项 (title="新对话")
  → 自动选中、清空消息流、输入框聚焦
```

#### 切换
```
点击列表项
  → 若有未结束 SSE，先 abort 并提示
  → GET /ai/chat/sessions/{id}
  → 渲染 messages 历史（按 createdAt 升序）
  → 滚动到底部
```

#### 删除
```
点击 🗑
  → ElMessageBox.confirm("确定删除此会话？删除后无法恢复")
  → DELETE /ai/chat/sessions/{id}
  → 从列表移除
  → 若是当前会话：选中第一项；列表为空显示空态
```

### 5.3 三态卡片

| 态 | 视觉 | 文案 / 图标 |
|---|---|---|
| 空态 | 居中插画 + CTA 按钮 | 🤖「还没有对话，点击开启第一次交流」按钮：开始新对话 |
| 加载中 | 3 个 skeleton 行 | 灰色脉冲动画 |
| 加载失败 | 中心红色感叹号 + 重试按钮 | "加载失败，[重试]" |

## 6. 消息流可视化语言

### 6.1 用户气泡

```
                                ┌──────────────────┐
                                │ 查询 PLAN-20260608│
                                │ -001 的状态       │
                                └──────────────────┘ 🙂 ← 头像首字母
                                14:30  ← hover 显示
```
- 右对齐，最大宽度 70%
- 背景 `#2c5aa0` / 文字白色 / `border-radius: 12px 12px 4px 12px`

### 6.2 AI 助手气泡

```
✦  ┌─────────────────────────────────────┐
   │ 当前有 3 个运行中的生产计划：        │
   │ 1. PLAN-20260608-001 焊接组件       │
   │ 2. PLAN-20260609-002 装配组件       │
   │ 3. ...                              │
   │                                     │
   │ ↑520 ↓210 总 730 tokens             │ ← token 信息
   └─────────────────────────────────────┘
```
- 左对齐，最大宽度 80%
- 背景 `var(--ai-bubble-bg)` / 边框 `1px solid var(--border-default)`
- 圆角 `12px 12px 12px 4px`
- 内容 Markdown 渲染（代码块、表格、列表）
- 底部 token 信息小字 12px 灰色

### 6.3 thinking 三点态

```
✦  ┌─────────┐
   │ ● ● ●  │   循环呼吸
   └─────────┘
```
- 仅在尚未收到任何 `text_delta` 之前显示
- 收到首个 `text_delta` 即替换为流式文本

### 6.4 text_delta 流式打字机

```
✦  正在为您查询运行中的计划▍
```
- 文本逐字符 append，光标 ▍ 闪烁
- `done` 事件后光标消失

### 7.1 自适应高度
- min-height 40px / max-height 160px
- 内容超出时显示纵向滚动条

### 7.2 字符计数
- 实时 `{count}/4000`
- count > 3800 黄色警告
- count > 4000 红色 + 输入框红边 + 发送按钮 disabled

### 7.3 按钮组状态切换

| 会话状态 | 发送按钮 | 清空按钮 |
|---|---|---|
| 空闲 | 📤 发送 (primary) | 🗑 清空 |
| 思考中 (SSE 流中) | ⏹ 停止 (warning) | 🗑 清空 (disabled) |
| 输入超长 | disabled | 🗑 清空 |

### 7.4 键盘交互

| 键 | 行为 |
|---|---|
| `Enter` | 发送（仅当未超长且非"思考中"） |
| `Shift + Enter` | 换行 |
| `Esc` | 关闭抽屉（同时 abort SSE） |
| `Ctrl + L` | 清空当前输入框 |
| `Ctrl + /` | 切换抽屉开/关 |

## 8. SSE 事件流时序与状态机

### 8.1 事件接收 → 视觉响应时序图

```
用户        前端 UI            SSE 流                后端
 │             │                  │                    │
 │ 发送消息    │                  │                    │
 │────────────▶│  POST /messages │                    │
 │             │─────────────────▶│                    │
 │             │                  │ event: thinking    │
 │             │  显示 ● ● ●     │◀───────────────────│
 │             │                  │ event: text_delta  │
 │             │  开始打字机      │◀───────────────────│
 │             │                  │ event: text_delta  │
 │             │  追加字符        │◀───────────────────│
 │             │                  │ event: tool_call   │
 │             │  插入工具卡(🔄) │◀───────────────────│
 │             │                  │ event: tool_result │
 │             │  工具卡(✅)     │◀───────────────────│
 │             │                  │ event: text_delta  │
 │             │  继续打字        │◀───────────────────│
 │             │                  │ event: done        │
 │             │  光标消失/token │◀───────────────────│
 │             │  恢复输入框      │                    │
 │◀────────────│                  │                    │
```

### 8.2 三级状态机

#### 会话级
```
[Idle] ─send─▶ [Streaming] ─done/error/abort─▶ [Idle]
                    │
                  switch session
                    ▼
                 [Aborted]
```

#### 消息级（每个 AI assistant 消息）
```
[Created] ─thinking─▶ [Thinking] ─text_delta─▶ [Streaming]
                                                    │
                                          tool_call │
                                                    ▼
                                            [ToolPending]
                                                    │
                                          tool_result
                                                    ▼
                                                [Streaming]
                                                    │
                                                  done
                                                    ▼
                                                 [Done]
```

#### 工具调用级
```
[Pending] ─tool_result(success)─▶ [Success]
   │     ─tool_result(failure)─▶ [Failure]
   │     ─session abort       ─▶ [Cancelled]
```

### 8.3 异常路径

| 场景 | 检测方式 | UI 反馈 |
|---|---|---|
| 网络断开 | `fetch` reject / reader 抛错 | 错误卡片「连接中断，请检查网络」+ 重试按钮 |
| 后端 5 分钟超时 | 长时间无 event 且 reader 关闭 | 错误卡片「响应超时」+ 重试 |
| 用户主动停止 | 调用 `AbortController.abort()` | AI 气泡尾部追加灰色斜体「（已中断）」 |
| 401 未授权 | response.status === 401 | 清除 localStorage user，跳转 `/login` |
| 4000 超长拦截 | 客户端 pre-check | 不发出请求，输入框红边提示 |

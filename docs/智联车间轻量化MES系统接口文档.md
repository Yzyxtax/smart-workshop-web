# 智联车间轻量化MES系统 接口文档

## 一、系统概述

智联车间轻量化MES系统是一套面向生产车间的制造执行系统，提供生产计划管理、订单管理、工艺流程管理、设备管理、班组管理等功能模块。系统采用前后端分离架构，后端基于Spring Boot框架构建，提供RESTful API接口。

### 1.1 技术架构

- **后端框架**：Spring Boot 3.5.7
- **持久层**：MyBatis
- **数据库**：MySQL
- **认证方式**：JWT Token
- **返回格式**：统一JSON格式

### 1.2 接口基础信息

- **基础路径**：`/api`（需确认具体配置）
- **响应格式**：

```json
{
  "code": 200,
  "message": "success",
  "data": {}
}
```

- **错误码说明**：
  - `200`：成功
  - `500`：服务器内部错误
  - `403`：权限不足

### 2.2 用户管理模块

#### 2.2.1 条件分页查询用户

| 项目       | 说明            |
| -------- | ------------- |
| **接口路径** | `GET /user`   |
| **功能描述** | 支持多条件分页查询用户列表 |

**请求参数说明**：

| 参数名      | 类型      | 必填 | 说明                    |
| -------- | ------- | -- | --------------------- |
| page     | Integer | 是  | 页码（默认1）               |
| pageSize | Integer | 是  | 每页数量（默认10）            |

| end      | String  | 否  | 创建日期结束（格式：yyyy-MM-dd） |

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "total": 100,
    "list": [
      {
        "id": 1,
        "userName": "admin",
        "name": "管理员",
        "position": "主管",
        "permissionLevel": "ADMIN"
      }
    ]
  }
}
```

**路径参数说明**：

| 参数名    | 类型         | 必填 | 说明   |

**动作枚举值说明**：

| 枚举值             | 说明   |
| --------------- | ---- |
| PUBLISH         | 发布   |
| CANCEL\_PUBLISH | 取消发布 |
| PAUSE           | 暂停   |
| RESUME          | 恢复执行 |
| START\_WORK     | 开始作业 |
| FINISH\_WORK    | 完成作业 |

**状态流转说明**：

| 当前状态         | 可执行动作           | 目标状态          |
| ------------ | --------------- | ------------- |
| CREATED（创建）  | PUBLISH         | RELEASED（发布）  |
| RELEASED（发布） | CANCEL\_PUBLISH | CREATED（创建）   |
| RELEASED（发布） | START\_WORK     | RUNNING（执行）   |
| RUNNING（执行）  | PAUSE           | PAUSED（暂停）    |
| RUNNING（执行）  | FINISH\_WORK    | COMPLETED（完成） |
| PAUSED（暂停）   | RESUME          | RUNNING（执行）   |
